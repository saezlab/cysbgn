This is a project to integrate CNOR with Cytoscape for visualization proposes using the SBGN standard notation. 

This will be done with two different plugins: 

-	SBGNplugin will enable import, export and visualize SBGN diagrams;
-	CNOplugin will be responsible for integrate CNOR with Cytoscape through a friendly interface.
